Setup instructions
  1. Run the Google Calendar Python quickstart on your local machine to get token.json.
  2. Open token.json and paste its full content below.
  3. Optionally paste your credentials.json (from Google Cloud Console) — needed if token.json doesn't include client_id/client_secret.

Microsoft 365

Business Agent only. Calendar, Outlook email, To Do, and OneNote.

Not configured
Setup instructions
  1. In Azure Portal, go to App registrations → your app → copy the Application (client) ID and Directory (tenant) ID.
  2. Under Certificates & secrets, create a client secret and copy it.
  3. Use the OAuth Device Code Flow or your existing OpenClaw refresh token to get a refresh_token.
  4. Required scopes: Calendars.ReadWrite Mail.Read Tasks.ReadWrite Notes.Read

From your existing OAuth flow or OpenClaw setup.
